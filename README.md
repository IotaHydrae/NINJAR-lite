## <h2 align="center">介绍</h2>

<!-- 有关项目的简短介绍 -->

<p align="center"><strong>NINJAR (NINJAR Is Not Just A Reader)</strong> 作为一款掌上阅读器，但恐怕远远不止这样</p>

<p align="center">正如我们一开始演示的这样，通过外接无线键盘，它可以当作一台掌上终端来使用</p>

<p align="center">通过开源的的设计、软件资料，你完全可以重新定义这个设备的形态（这得益于高度模块化的设计)</p>

<p align="center">这个项目正处于早期开发阶段，我们正在尝试加入更多的创客功能，以支持模块拓展</p>

> <p align="center"><strong>NINJAR-lite</strong> 是基于全志 F1C100s 设计的精简版本</p>

## <h2 align="center">配置</h2>

<!-- 有关设备资源的表格 -->

| 组件       | 型号                          |
|------------|-------------------------------|
| 系统       | linux、Debian 12             |
| CPU        | Allwinner F1C100s/F1C200s ARM9 700MHz |
| 内存       | 32/64MB DDR1 256MHz              |
| 存储       | 128/256MB SPI-Nand Flash              |
| 屏幕       | 1.28英寸 16灰阶 OLED 128x128         |
| 网络       | 2.4G WiFi AP                  |
| 定位       | GP-02 GPS/BDS 模组            |
| 控制       | 拨轮编码器、按键               |
| 传感器     | AHT20                         |
| 电池       | 450mAh                        |
| 充放电 | TP4056 MOSFET                    |
| 接口 | SH1.0mm 4Pin I2C， USB 2.0 OTG, Dual Role |
