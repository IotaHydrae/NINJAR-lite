## 核心板

初始状态： 漏电 480uA

我们知道，PCB常见的漏电问题，有如下几个原因：

- 上拉电阻的大小
- DC-DC的静态电流
- LDO的静态电流

在本案例中，通过`POWER_EN`信号将三路DC-DC关闭后，那么板上就只有一个上拉电阻在耗电了
