## <h2>介绍</h2>

默认发货配置：

- NINJAR-lite 外壳套件 x 1
- NINJAR-lite 底板 x 1
- NINJAR-lite 核心板 x 1
- NINJAR-lite OLED 屏幕 x 1
- 450 mAh 锂电池

可选购配件：

- embeddedboys USB转串口模块

可能需要自己准备的配件：

- USB转串口模块
- USB Type-C 数据线 x 1
- USB Type-C OTG转接头 x 1
- 键盘、鼠标

`NINJAR-lite` 支持从如下存储介质启动：

- SPI-Nor Flash
- SPI-Nand Flash
